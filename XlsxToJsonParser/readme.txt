1. Для запуска потребуется среда исполнения .NET 9.
   Если она не установлена, скачать можно с официального сайта майкрософт
2. После того как вы убедились в наличии среды исполнения достаточно скопировать
   в папку с приложением файл "Тестовое задание.xlsx" и запусть файл XlsxToJsonParser.
   Если вы дополнительно хотите видеть сообщения об ошибках,
   тогда необходимо запустить программу из консоли
